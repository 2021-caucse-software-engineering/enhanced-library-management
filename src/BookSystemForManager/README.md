## 실행 방법

백엔드 서버는 현재 AWS EC2에서 구동되고 있습니다.

1. front-end 디렉토리로 이동

2. 필요한 Dependency를 설치한다.

```bash
yarn # or yarn init
```

3. 리액트 프로젝트를 실행한다.

```bash
yarn start
```

4. 로그인하여 서비스를 사용한다.
